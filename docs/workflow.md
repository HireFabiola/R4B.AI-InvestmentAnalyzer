# Workflow

The MVP workflow follows the path of a basic buy-and-hold rental acquisition review.

## Planned Flow

1. User enters property and investment assumptions.
2. Property Intake Agent validates and organizes the inputs.
3. Market & Comps Agent reviews available market context.
4. Rehab & Condition Agent reviews condition and repair assumptions.
5. Financial Analysis Agent runs deterministic rental calculations.
6. Investment Recommendation Agent produces a final recommendation.

## Workflow Principles

- Keep each step focused.
- Use structured data between agents.
- Keep financial calculations deterministic and testable.
- Make recommendations explainable for beginners.
- Clearly flag missing information instead of hiding uncertainty.
